##  Landing based on webpack
Landings page with GSAP Tweenmax, Clipboard.js, jQuery functionality based on Webpack 

## Requirements

-   nodejs 8+
-   npm 6+
-   yarn 1.15 + `npm i -g yarn`

## Usage

Download all dependencies:

```bash
cd path/to/webpack-project-backbone
```

```bash
yarn
```

For development:

```bash
yarn start
```

For deployment:

```bash
yarn build
```
