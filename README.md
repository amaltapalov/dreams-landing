-   yarn 1.15 + `npm i -g yarn`

## Usage

Download all dependencies:

```bash
cd path/to/webpack-project-backbone
```

```bash
yarn
```

For development:

```bash
yarn start
```

For deployment:

```bash
yarn build
```
